Coloque aqui as imagens do Hero (background) que quer usar no site.

Nome recomendado e ordem (substitua os arquivos por suas fotos):
- `hero-1.jpg` (desktop principal)
- `hero-2.jpg` (opcional)
- `hero-3.jpg` (opcional)

Recomendações de formato/tamanho:
- Formato: `jpg` ou `png` (jpg preferível para fotos)
- Tamanho: largura mínima 1920px para boa qualidade em desktop
- Qualidade: mantenha a compressão para web (ex.: 60-80%)

Observações:
- Após adicionar/renomear as imagens, apenas recarregue http://localhost:3000/ — o Vite serve arquivos da pasta `public/` automaticamente.
- Se quiser usar nomes diferentes, atualize o array `localImages` em `components/Hero.tsx`.
